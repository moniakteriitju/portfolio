Project 1:

Project Title: Regional Covid-19-Tracking System.
A website to provide covid information
Duration: 3 months.
Technologies Used: HTML,CSS,PHP,MySQL.

Features:

1. A database is maintained to keep the information and SQL is used to manipulate the stored data.
2. 3 types of user of this website:
Doctor
User
Admin
3. Doctors will be able to see patient overviews under his observation. Also he can search for overviews in his hospital.
4. User can find out there necessary information both by reading the static overview or can manually search for a result.
5. Admin can update, delete or insert data into the database.

Project 2:

Project Title: Animal Care
A website to provide services to Animal lover
Duration: 15 days.
Technologies Used: HTML,CSS,PHP,MySQL.

Features:

1.Doctors can register and log into the website.
2. Members who are interested can register and log into the website.
3. Admin panel will manage all data in the back end.
4. Logging in as a member, a person can perform the following task:
- Can see all previous help-seeking posts.
- Can create new help-seeking posts.
- Can visit his/her own profile.
- Can edit his/her profile.
- Can upload profile picture.
- Can write comment and see other members comments.
- Can see previous activity.
- Can search available doctors info.
5. Doctor Activities in our website:
- Can see on profile.
- Can edit profile.
- upload profile picture.
- Can visit homepage and see posts.
6. Admin Activities in our website:
- Can delete any data from the website.
- Manage access to the website.
- Can change his password.
- Can register new admins.
- Can delete any post from the home page.
- Can restrict the access of any member.
